# PWA Icons

To complete the PWA setup, you need to add icon files to the `public/` directory:

1. `icon-192x192.png` - 192x192 pixels
2. `icon-512x512.png` - 512x512 pixels

You can:
- Use an online icon generator (e.g., https://realfavicongenerator.net/)
- Create icons with your app logo/design
- Use a simple placeholder icon for development

The icons should be square PNG images with your app's branding.
